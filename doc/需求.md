# 需求概述

- 使用.NET 10 + WPF，编写一个i18n 字符串资源文件编辑器，名为I18nStringEditor。
- 字符串资源以json格式保存，示例如下：
```json
{
  "Common": {
    "Ok": "确定",
    "Cancel": "取消",
    "Error": "错误"
  },
  "MainWindow": {
    "Title": "我的超级应用",
    "Welcome": "欢迎使用 Quicker3",
    "ToggleTheme": "切换主题",
    "Menu": {
      "File": "文件",
      "Edit": "编辑",
      "Test": "测试"
    }
  },
  "Settings": {
    "Theme": "主题",
    "Language": "语言"
  },
  "Areas": {
    "Accounts": {
      "AccountSelectionWindow": {
        "Title": "选择账户",
        "Prompt": "请选择一个账户以继续："
      },
      "Common": {
        "LocalAccountTitle": "本地账户",
        "LocalAccountDesc": "离线模式，数据仅保存在本地。"
      }
    }
  }
}
```

## 功能概述
- 程序启动时，自动打开上一次编辑的文件。 在标题栏中显示文件路径。
- 在资源文件相同位置，保存 "资源文件名.info" 文件，用以存储如下信息：
    - 节点的展开折叠状态。
    - 节点的注释。
    - 全局设置：
        - StringKey模版，用以根据Key生成在XAML中使用的文本。
- 为左右结构，左侧为父节点的节点树，右侧为此节点下的文本列表。 
    - 左侧：节点树，支持展开折叠每个节点、为节点添加子节点、排序节点等操作。
    - 右侧：分为上下个面板。
        - 上部面板（高）：用于编辑当前父节点下的文本资源列表，分为3列：Key、值、说明。（字符串说明信息保存在info文件中）。
        - 下部面板（矮）：显示同目录下，其它语言中本字符串的值。 界面中增加按钮，用以选择是否显示此面板。 
- 支持搜索。
- 支持快捷键：
    - 快速创建字符串资源。
    - 快速生成并复制用以在XAML中使用的StringKey文本，如`{I18N {x:Static Strings.Areas_Accounts_Common_LocalAccountDesc}}`,用以快速填写到TextBlock的Text值中。
- 修改后自动保存到资源文件。
- 其它必要的功能，如“打开”按钮等。
